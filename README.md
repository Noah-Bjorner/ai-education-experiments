# Edu Experiments API

Deno + TypeScript + Hono API for small education experiments.

## Commands

```sh
deno task dev
deno task start
deno task check
deno task lint
deno task fmt
```

The server listens on `PORT` when set, otherwise `8000`.

## Demo Endpoints

- `GET /`
- `GET /health`
- `GET /demo`
